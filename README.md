## 🚀 Featured Project

### 🔥 GPCA-Augmented HazDesNet for Haze Density Estimation

A deep learning model that estimates haze density from images using **Gaussian Process Channel Attention (GPCA)** integrated with **HazDesNet**.

🔗 Project Link: https://github.com/gurulakshmi2909/hazdesnet-gpca.git

### ✨ Key Features:

* 📷 Image-based haze density prediction
* 🧠 Deep learning using TensorFlow
* 🎯 Gaussian NLL loss for probabilistic regression
* 📊 Outputs both **haze density + uncertainty**
* 🔍 Attention mechanism for improved feature learning
